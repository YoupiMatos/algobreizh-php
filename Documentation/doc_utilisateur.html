Bienvenue sur la documentation utilisateur d'AlgoBreizh.

Ce document vous permettra d'utiliser facilement le site AlgoBreizh.


Acceuil

La page d'accueil d'algobreizh. De cette page, vous pouvez vous connecter (1) ou vous inscrire (2) en cliquant sur les boutons de la barre de navigation.


Inscription

Sur la page d'inscription, renseignez vos informations. Cliquez ensuite sur "S'incrire".

Si l'inscription à réussi, vous verrez cette page :


Connexion

Une fois inscris, dirigez vous vers la page de connexion.

Renseignez votre nom d'utilisateur et votre mot de passe, puis cliquez sur "Se connecter".

Si la connexion à réussi, vous verrez cette page :

Vous constaterez alors que la barre de navigation à changée. Elle permet dès lors d'utiliser les fonctions du site :

(1) Voir vos commandes, (2) voir vos factures, (3) saisir une nouvelle commande et (4) vous déconnecter.


Voir ses commandes

Cliquez sur le bouton "Mes commandes". Une liste de toutes vos commandes non confirmées s'affiche.

Vous pouvez cliquer sur "Détails de la commande" pour voir les détails d'une commande choisie.

Cette page vous permet de voir les articles commandés, leur quantité, le prix total de la commande ainsi que le statut de la commande : confirmée ou non.


Voir ses factures

Cliquez sur la bouton "Mes factures". Une liste de toutes vos commandes confirmées par un administrateur s'affiche.

Vous pouvez voir le détail de ses commandes validées de la même façon qu'avec les commandes non validées.


Nouvelle commande

Cliquez sur le bouton "Nouvelle commande". Une page s'affiche, avec les différents produits disponibles.

Renseignez la quantité de chaque produit souhaité, puis cliquez sur "Confirmer".

Si la commande à réussi, vous verrez cette page :

Vous pourrez alors la retrouvez dans la rubrique "Mes commandes".
